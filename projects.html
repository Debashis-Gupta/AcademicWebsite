Credit Card Fraud Detection

Technology Can Be Curse When It Is Being Abused

View on GitHub

It is important that credit card companies are able to recognize fraudulent credit card transactions so that customers are not charged for items that they did not purchase. So I tried on credit card fraud detection dataset from Kaggle to detect the fraud cases. It is basically a categorical implementation to detect normal and fraud transactions.

Here I implemented three algorithms IsolationForest, LocalOutlierFactor, RandomForest and Artificial Neural Network (ANN). The highlighted remarks of this project is:

  • Comparison between static algorithms and neural network architectural algorithms
  • ANN performs better than IsolationForest and RandomForest
  • Dataset Analysis

Keylogger Attack

Understanding Security Vulnerabilities

View on GitHub

A keylogger, sometimes called a keystroke logger or system monitor, is a type of surveillance technology used to monitor and record each keystroke typed on a specific computer's keyboard. Intruders use keyloggers to steal personal information which they can then sell or use for own profit.

I made this keylogger attack using Python script. After running it, any keystroke can be traced by the script automatically. The key features of it:

  • All keystrokes are traced and note down on a text file
  • This keylogger works behind the any app automatically
